What is HTML?

HTML stands for HyperText Markup Language. Think of it as the skeleton of a webpage - it gives structure to your content.

Real-world analogy:

If a webpage is like a building:

  • HTML = The basic structure (walls, rooms, doors)
  • CSS = The interior design (colors, furniture, decoration)
  • JavaScript = The electrical system (lights, fans, interactive elements)

HTML Tags - The Building Blocks

HTML uses tags to mark up content. Tags are like labels that tell the browser "this is a heading", "this is a paragraph", etc.

<tagname>Content goes here</tagname>

Tag Structure:

  • Opening tag: <tagname>
  • Content: The text or other elements
  • Closing tag: </tagname> (notice the forward slash)

Basic HTML Structure

Every HTML page has the same basic structure:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <title>Page Title</title> </head> <body> <h1>My First Heading</h1> <p>My first paragraph.</p> </body> </html>

What each part does:

  • <!DOCTYPE html> - Tells browser this is HTML5
  • <html> - The root element, contains everything
  • <head> - Information about the page (not visible to users)
  • <title> - Shows in browser tab/window title
  • <body> - The visible content users see

HTML Attributes - Adding Extra Information

Attributes provide additional information about HTML elements. They go inside the opening tag.

<img src="logo.png" alt="Company Logo" width="200"> <a href="https://www.google.com" target="_blank">Google</a> <input type="email" id="email" placeholder="Enter your email">

Common attributes:

  • src - Source file for images
  • href - Destination URL for links
  • id - Unique identifier for an element
  • class - Group identifier for styling
  • alt - Alternative text for images
  • placeholder - Hint text in form fields
